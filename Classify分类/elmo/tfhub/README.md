# 前言
    该版本是利用tf.hub中的elmo 进行文本分类，模型的输入是文本，数据形式
# tensorflow_hub安装
    pip install tensorflow_hub
# 运行环境    
    详见requirements.txt(注意：tf.hub 需要tensorflow>=1.15)
# 代码执行
    注：可单独只下载tfhub目录下的文件，就可以执行
    1、进入tfhub,运行main.py
# tensorflow_hub使用
可以参考官网[tensorflow_hub 官网](https://tensorflow.google.cn/hub/overview)
